# Hlyvka Kate — 3D Visualizer Portfolio

Vite + React односторінковий сайт портфоліо.

## Запуск

```bash
npm i
npm run dev
```

Після запуску відкрий адресу, яку покаже Vite, зазвичай:

```bash
http://localhost:5173
```

## Де редагувати

- Основна сторінка: `src/App.jsx`
- Глобальні стилі Tailwind: `src/index.css`
- Tailwind config: `tailwind.config.js`

## Примітка

У цьому архіві Tailwind зафіксований на версії 3.x, щоб PostCSS-конфігурація працювала через стандартний плагін `tailwindcss` без помилки Tailwind v4.
